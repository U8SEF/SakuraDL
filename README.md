## ✨ Features

* 🌸 Download anime episodes with ease
* 🎬 Multiple quality selection (1080p / 720p / 480p)
* ⚡ High-speed downloads powered by **aria2c**
* 🎨 Beautiful terminal user interface
* 🔍 Automatic anime and episode information extraction
* 📥 Simple and lightweight workflow
* 🐍 Built entirely with Python

---

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/SakuraDL.git
cd SakuraDL
```

Install the required Python packages:

```bash
pip install -r requirements.txt
```

Make sure **aria2c** is available inside the `bin` directory (or in your system PATH).

Run SakuraDL:

```bash
python main.py
```
